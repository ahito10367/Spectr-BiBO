# Spectr-BiBO
Расчет распределения СПР(спонтанного параметрического рассеяния) для подбора идеального угла в данном случае для BBO (второй тип). Для других кристаллов необходимо заменить показатели преломления. При успользовании других параметров падающего на кристалл излучения необходимо изменить длину волны.
# Spectral Signal Analysis: Peak Detection & Feature Extraction

> Инструмент для обработки спектральных данных: фильтрация шумов, детекция пиков, извлечение признаков.  
> Демонстрирует навыки работы с сигналами — применимо в финтехе (временные ряды), телекоме (обработка сигналов), MedTech (ЭКГ/ЭЭГ), IoT (мониторинг оборудования).

## Для чего это может быть полезно в Data Science
- **Обработка сигналов**: фильтрация (Savitzky-Golay, медианная), удаление базлайна, нормировка — аналог предобработки временных рядов
- **Детекция событий**: поиск пиков, оценка ширины/амплитуды/площади — аналог детекции аномалий в транзакциях или сетевом трафике
- **Извлечение признаков**: расчёт спектральных метрик  — прямой аналог feature engineering для ML
- **Визуализация**: построение спектров с аннотациями, сравнение условий — ключевой навык для презентации стейкхолдерам

## Стек
- **Язык**: Python 3.x
- **Библиотеки**: `numpy`, `scipy.signal`, `pandas`, `matplotlib`, `scikit-learn` (опционально)
